# 데이터 분석 [PCCE 기출문제] 10번

## 문제 설명
AI 엔지니어인 현식이는 데이터를 분석하는 작업을 진행하고 있습니다. 데이터는 ["코드 번호(code)", "제조일(date)", "최대 수량(maximum)", "현재 수량(remain)"]으로 구성되어 있으며 현식이는 이 데이터들 중 조건을 만족하는 데이터만 뽑아서 정렬하려 합니다.

## 입력 형식
- data: 2차원 정수 리스트 (1 ≤ 길이 ≤ 500)
  - data[i]: [코드 번호, 제조일, 최대 수량, 현재 수량] 형태
  - 1 ≤ 코드 번호 ≤ 100,000
  - 20000101 ≤ 제조일 ≤ 29991231 (yyyymmdd 형태)
  - 1 ≤ 최대 수량 ≤ 10,000
  - 1 ≤ 현재 수량 ≤ 최대 수량
- ext: 문자열 ("code", "date", "maximum", "remain" 중 하나)
- val_ext: 정수 (ext에 따른 올바른 범위의 값)
- sort_by: 문자열 ("code", "date", "maximum", "remain" 중 하나)

## 출력 형식
- ext 값이 val_ext보다 작은 데이터만 추출하여 sort_by 기준으로 오름차순 정렬한 2차원 리스트

## 제약 조건
- 조건을 만족하는 데이터는 항상 한 개 이상 존재
- 정렬 기준에 해당하는 값이 서로 같은 경우는 없음

## 예시 입력/출력
입력:
data = [[1, 20300104, 100, 80], [2, 20300804, 847, 37], [3, 20300401, 10, 8]]
ext = "date"
val_ext = 20300501
sort_by = "remain"

출력:
[[3, 20300401, 10, 8], [1, 20300104, 100, 80]]
