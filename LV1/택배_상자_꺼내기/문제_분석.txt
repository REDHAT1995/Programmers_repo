# 택배 상자 꺼내기 문제 분석

## 문제 접근 방법
1. 상자를 쌓는 패턴 분석
   - 왼쪽에서 오른쪽으로 쌓기
   - 오른쪽에서 왼쪽으로 쌓기
   - 이 패턴이 반복됨

2. 상자 위치 계산
   - 각 상자의 위치는 층과 열로 표현 가능
   - 층은 0부터 시작
   - 열은 0부터 w-1까지

3. 알고리즘 설계
   - 주어진 번호의 상자가 몇 층, 몇 열에 있는지 계산
   - 해당 상자 위에 있는 모든 상자의 개수 계산

## 시간/공간 복잡도
- 시간 복잡도: O(n)
- 공간 복잡도: O(1)

## 최적화 포인트
1. 상자 위치 계산 최적화
   - 수학적 공식을 사용하여 직접 계산
   - 불필요한 반복문 제거

2. 메모리 사용 최적화
   - 2차원 배열 대신 수학적 계산 사용
   - 상수 공간만 사용

## 고려사항
1. 경계 조건
   - n이 w의 배수인 경우
   - 마지막 층이 완전하지 않은 경우

2. 예외 처리
   - 입력값 범위 검증
   - 잘못된 입력 처리
