# 유연근무제 문제 분석

## 문제 접근 방법
1. 요일 처리
   - startday를 기준으로 평일/주말 구분
   - 주말(토,일)은 체크하지 않음
   - 요일 순환 처리 필요

2. 시간 비교
   - 시각을 분 단위로 변환하여 비교
   - 출근 희망 시각 + 10분까지 허용
   - 시간 형식 변환 (HHMM → 분)

3. 알고리즘 설계
   - 각 직원별로 평일 출근 기록 확인
   - 모든 평일에 지각하지 않은 직원 카운트
   - 결과 반환

## 시간/공간 복잡도
- 시간 복잡도: O(n * 7) = O(n)
- 공간 복잡도: O(1)

## 최적화 포인트
1. 시간 변환 최적화
   - 시각을 분으로 변환하는 함수 최적화
   - 불필요한 변환 제거

2. 요일 처리 최적화
   - 요일 체크 로직 단순화
   - 주말 판별 최적화

## 고려사항
1. 경계 조건
   - 주말 처리
   - 시간 형식 변환
   - 요일 순환

2. 예외 처리
   - 입력값 범위 검증
   - 시간 형식 검증
   - 배열 길이 검증
